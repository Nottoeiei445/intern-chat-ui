### Tech Stack ที่ใช้
```text
Framework: Next.js
Language: TypeScript
Styling: Tailwind CSS
UI Component: Shadcn UI
Package: Bun
AI & LLM: Ollama

```

### วิธีรันโปรเจกต์ (AI Chat)
1. clone โปรเจกต์
```
ไปที่ cmd
git clone (URL)
```

2. ติดตั้ง Library:
```bash
bun install
```

3. เริ่มรันโปรเจกต์:
```bash
bun dev
```
